## Image Guidelines
- **Format**: PNG for diagrams/screenshots, JPG for photos
- **Size**: Optimize to <500KB each for fast loading
- **Resolution**: Minimum 1200px width for diagrams, 800px for features
- **Naming**: Use exact filenames as specified in HTML files

## Taking Dashboard Screenshots
1. Open https://harisivasaiteja.github.io/Studentdashboard/
2. Set browser window to 1920x1080 resolution
3. Use screenshot tool (Windows: Snipping Tool, Mac: Cmd+Shift+4)
4. Capture main dashboard view with KPIs and charts
5. Navigate to a student profile and capture that view
6. Save as dashboard-main.png and dashboard-profile.png
7. Optimize file sizes using tinypng.com if needed
