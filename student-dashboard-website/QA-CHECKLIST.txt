Issue: Images not loading
Solution: Check file paths are relative (./images/file.png), verify files exist

Issue: Dark mode not persisting
Solution: Check browser localStorage enabled, try clearing cache

Issue: Mobile menu not working
Solution: Verify JavaScript loaded, check for console errors

Issue: Layout broken on mobile
Solution: Test viewport meta tag, check media queries

Issue: GitHub Pages not updating
Solution: Clear GitHub cache, wait 5-10 minutes, force refresh (Ctrl+Shift+R)

Issue: Fonts not loading
Solution: Check internet connection (Google Fonts require online access)
